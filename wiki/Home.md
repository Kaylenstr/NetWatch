# NetWatch

Network monitoring dashboard. Monitors servers via ICMP ping and shows:

- Online/offline status
- Latency (ms)
- Jitter (ms)
- Map view with server locations
- Latency and jitter charts over time

Servers are configured in `servers.json` or via the Settings page in the web UI. Changes take effect immediately.
